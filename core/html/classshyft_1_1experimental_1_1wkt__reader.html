Detailed Description

wkt_reader, WellKnownTextformat(?) reader: Just for the testing and experimental purposes,we need something that gives a reasonable amount of data that is realistic. From Statkraft we have the nea-nidelv system, and this reader is used to parse/read those files into memory/structures that we can use to construct cells, environmental time-series, etc. that are needed to run a full model. (and GIS system might provide 'non-perfect' data that we need to handle)

Note
this class contains readers for the files located in test/neanidelv directory
