Detailed Description

template<class PS>
struct shyft::core::model_calibration::target_specification< PS >

The target specification contains:

  1. a target ts (the observed quantity)
  2. a list of catchment ids (zero-based), that denotes the catchment.discharges that should equal the target ts.
  3. a scale_factor that is used to construct the final goal function as.

goal_function = sum of all: scale_factor*(1 - nash-sutcliff factor) or KLING-GUPTA

Template Parameters
PStype of the target time-series, any type that is time-series compatible will work. Usually a point-based series.
