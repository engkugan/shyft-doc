Bayesian Temperature Kriging Interpolation.

Extracted from the Enki method BayesTKrig by Sjur Kolberg/Sintef.

Scatters a set of time-series data at given source locations (X,Y,Z) to a new set of time-series data at target locations (X,Y,Z) using a Bayesian Temperature Kriging method where the vertical distances used during the semivariogram's covariance calculations are scaled with a height factor.

In addition to interpolating values at the destinations, estimates on the sea level temperature and temperature gradients are computed at each time step.

Preconditions:

  1. There should be at least one source location with a valid value for all t.
